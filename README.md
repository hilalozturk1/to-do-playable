## Getting Started

First, run backend:

```bash
npm run start
# or
yarn start
# or
pnpm start
# or
bun start
```

Next, run frontend:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3001](http://localhost:3001) 

You can log in with this username and password. 
=> user1 password1 or user2 password2.
